# Signal Lantern

## App Name

Signal Lantern

## Tagline

Share your live signal

## Description

Publish a public status lantern with state, color, message, and active days so people can see what you are doing on Base.

## Category

Social / identity / status

## Short Pitch

Signal Lantern gives builders and creators a simple onchain status light: what they are doing, how long it is active, and a short message others can read.

## Production Environment Variables

- `NEXT_PUBLIC_SIGNAL_LANTERN_CONTRACT_ADDRESS`: deployed Signal Lantern contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
